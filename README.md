# Code-alpha-project
Project 2
I created a cybersecurity awareness training module on Phishing Attacks as part of my learning journey at CodeAlpha.
This presentation explains how to recognize phishing emails and fake websites, understand social engineering tactics, and follow best practices to avoid becoming a victim of cyber attacks.
The module also includes real-world examples and interactive quiz questions to make learning more engaging.
Cybersecurity starts with awareness — Think before you click!
